# secur_audit
Курсовой проект клиент-серверное приложение для аудита безопасности сетевых сервисов, проверяет с помощью поключения к TCP портам, но только с 1 по 1000, для того чтобы уменьшить время работы при аудите, но при этом оставлять покрытыми большинство популярных сайтов таких как google.com, spotify.com и др.
